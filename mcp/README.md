# MCP Layer — Omega Singularity

Model Context Protocol: centrale hub voor externe tools.

## Geplande servers

- **Google Workspace** — Drive, Sheets, Gmail
- **GitHub** — repos, issues, PRs
- **Brave Search** — market trends, proactieve scan
- **Filesystem** — lokaal bestanden

Configureer in Cursor / MCP-settings en via `.env` (zie `.env.example`).
